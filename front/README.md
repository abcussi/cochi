### Iniciar el fornt 
* entrar a la carpeta front 
* Instalar las dependencias
* Usar el comando "start"
### Start API server
* Clonar este repositorio
* Instalar las dependencias
* Usar el comando "serve"

* llenar el fomrulario con las validaciones 
* probar el reset

### Buildear 
* Usar el comando "build"
* Montar en un server
